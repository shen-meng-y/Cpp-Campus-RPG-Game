# campusRPG 校园冒险养成游戏

## 项目简介

`campusRPG` 是一个基于 **Qt / C++** 开发的校园主题 RPG 养成游戏。项目以大学校园生活为背景，将角色养成、背包道具、商店购买、任务系统、战斗探索等内容结合起来，玩家可以创建角色、提升等级、购买物品、完成任务，并在校园地图中探索和战斗。

本项目主要用于程序设计课程设计实践，重点练习 **Qt 图形界面开发、C++ 面向对象设计、数据持久化、游戏状态管理以及多模块协作开发**。

---

## 主要功能

### 1. 角色系统

- 支持创建角色
- 显示角色基本状态
  - 姓名
  - 等级
  - HP 生命值
  - EXP 经验值
  - 金币
  - 攻击力
  - 击败敌人数
- 支持角色升级与属性变化
- 支持 HP 和 EXP 进度条显示

### 2. 背包系统

- 支持查看当前拥有的物品
- 支持使用道具
- 支持物品数量管理
- 与商店、战斗奖励等模块进行联动

### 3. 商店系统

- 支持购买物品
- 购买时扣除金币
- 购买成功后物品加入背包
- 可用于补充恢复类道具或其他游戏资源

### 4. 任务系统

当前任务系统支持基础任务逻辑，并计划进一步优化为更完整的任务流程：

- 查看任务
- 接受任务
- 完成任务
- 领取奖励
- 区分不同类型任务，例如：
  - 主线任务
  - 日常任务
  - 成就任务
  - 探索任务

### 5. 战斗系统

- 支持进入战斗界面
- 支持校园地图探索
- 不同地点可能遇到不同怪物
- 支持选择怪物进行攻击
- 战斗中显示玩家和怪物的生命值
- 支持攻击、逃跑等基础战斗操作
- 战斗胜利后可获得经验、金币或任务进度

### 6. 存档系统

- 支持保存角色数据
- 支持读取已有存档
- 保存内容可包括：
  - 角色属性
  - 背包物品
  - 任务状态
  - 击败敌人数
  - 游戏进度

### 7. 地图与寻路

- 项目中尝试使用 A* 寻路算法
- 地图模块用于后续校园探索、碰撞检测和怪物刷新等功能扩展

## 运行环境

推荐环境：

- Qt 6.x
- C++17 或更高版本
- CMake
- MinGW / MSVC 编译器
- Windows 系统

开发环境示例：

```text
Qt Creator
Desktop Qt 6.x MinGW 64-bit
CMake 构建项目
```

---

## 运行方法

### 方法一：使用 Qt Creator 运行

1. 打开 Qt Creator
2. 选择 `Open Project`
3. 打开项目中的 `CMakeLists.txt`
4. 选择合适的 Kit，例如：

```text
Desktop Qt 6.x MinGW 64-bit
```

5. 点击构建按钮
6. 点击运行按钮启动游戏

### 方法二：使用命令行构建

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

构建完成后，运行生成的可执行文件即可。

## 课程设计说明

本项目作为程序设计课程设计作品，主要目标不是追求复杂商业游戏效果，而是通过一个完整的小型游戏项目，综合运用以下知识：

- C++ 类与对象
- Qt 信号与槽机制
- UI 控件设计
- 文件读写 / 数据库保存
- 多窗口交互
- 游戏逻辑封装
- 模块化开发
- Git 版本管理
